# Codigos_Simples_Java_Estudos
Alguns códigos simples em linguagem Java, realizados para estudo e prática.

- A pasta "contas" contém um sistema simples de criação e login de conta.
- A pasta "urna_eleitoral" contém um sistema simples que simula uma urna eleitoral.
- A pasta "letterboxd" contém um sistema simples de catálogo de filmes.
